**Air Quality Monitoring Dashboard** (4 Major cities-India) **Objective:** To visualise and compare 3 air quality parameters across four major Indian cities **(PM 2.5, PM 10, NO2- Delhi, Mumbai, Bengaluru, Kolkata)** with threshold-based interpretation and a map-based spatial overview. **Tools used:** Python, Streamlit, Pandas, Matplotlib, SQLite, Folium. **Live Dashboard** https://time-series-air-quality-stcvbuye7gnh28ak33mu7o.streamlit.app/
